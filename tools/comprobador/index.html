¿Por qué las letras "RA" no bastan para el marcador?

La cámara no "lee" el dibujo: busca puntos de anclaje (esquinas y detalles con contraste). Cada punto verde abajo es uno que el motor encontró. Más puntos = más fácil de reconocer. El piso para que enganche bien es ~1800.

Medición con el compilador real de MindAR (el mismo que usa la app). "Detección" = puntos de reconocimiento sumando todas las escalas.
